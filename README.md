# Magnifier Subway

Magnifier Subway is an Android app that helps users find the fastest subway routes in Seoul using real-time data. The app also provides features like route search, station information, and user-friendly UI/UX.

## Features
- Real-time subway route search
- Station information
- Integration with Firebase and push notifications

## Tech Stack
- Kotlin
- Firebase
- Android Jetpack Libraries (Room, LiveData, ViewModel, etc.)
- Retrofit for network requests

## How to Run
1. Clone the repository: `git clone https://github.com/1102june/Magnifier_Subway.git`
2. Open the project in Android Studio
3. Add `google-services.json` to the `app` directory
4. Run the app using an emulator or connected device
